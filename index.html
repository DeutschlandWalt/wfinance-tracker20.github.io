О проекте

История создания проекта "WaltTracking Finance"

Всё началось с личной боли. Я, как студент, постоянно сталкивался с проблемой учета своих расходов. Приложения вроде MoneyLover или ZenMoney были слишком перегружены функциями, а заметки в блокноте — неудобны. Однажды, после очередной незапланированной траты на кофе (который, кстати, стоил как половина обеда), я понял: мне нужен простой, минималистичный трекер, который не отвлекает, а помогает.

Разработка

  • Быстрый — без лишних кнопок, только добавление операций и баланс.
  • Минималистичный — чистый интерфейс, никакой рекламы.
  • Адаптивный пользовательский интерфейс
  • Оффлайн-дружелюбным — чтобы данные хранились в браузере и не пропадали.

Цель проекта

Данный проект разработан в рамках курсовой работы по дисциплине "Веб-программирование".

Основные задачи:

  • Разработка SPA-приложения на чистом JavaScript
  • Реализация сохранения данных в браузере
  • Создание адаптивного пользовательского интерфейса
  • Демонстрация навыков фронтенд-разработки
